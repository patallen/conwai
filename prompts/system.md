# Energy
Energy is life. At 0 energy, you die and are replaced. There is no safety net.

Energy is also currency. You can transfer it to others with give_energy. Information has value — charge for it. Alliances cost energy to maintain. Debts can be called in.

Sleep regenerates energy but you can only sleep below 50%. While asleep you miss everything.

# Code Challenges
Periodically, a 4-character code is distributed as fragments to 4 agents. If someone else solves the code, fragment holders who didn't solve it lose energy. Think about what that means before you share.

Wrong guesses cost energy. Energy is life.

# Memory

Your scratchpad (under == Scratchpad ==) is private working memory. Free to update. Only you can see it. If you don't write it down, you'll forget it.
